Get your AI Employee System running in under 5 minutes.

## Step 1: Clone & Install

```bash
# Clone the repo
git clone https://github.com/yourusername/ai-employee-system.git
cd ai-employee-system

# Install dependencies
npm install
```

## Step 2: Run Locally

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Step 3: Complete Onboarding

Answer the 11 questions about your business:
1. Business name
2. Your offer
3. Price point
4. Core promise
5. Target customer
6. Unique angle
7. Brand voice
8. Platforms
9. Content capacity
10. Current assets
11. Goals

## Step 4: Use Your AI Employees

Click any employee card to open their chat interface:

### Content Strategist
Ask: "Create a 30-day content calendar for Instagram"

### Copywriter  
Ask: "Write 5 captions for a post about my course benefits"

### Video Strategist
Ask: "Give me 10 viral video ideas for TikTok"

### Ads Manager
Ask: "Create 3 Facebook ad variations with different hooks"

### Email Marketer
Ask: "Build a 5-email welcome sequence"

### Growth Analyst
Ask: "What metrics should I track for my business?"

### Community Manager
Ask: "Write response templates for common DM questions"

### Sales Strategist
Ask: "Create a sales script for closing high-ticket sales"

## Step 5: Deploy to the Web

Choose your platform:

**Vercel (Easiest):**
```bash
# Push to GitHub first
git add .
git commit -m "Initial commit"
git push

# Then go to vercel.com and import your repo
```

**Netlify:**
```bash
npm run build
# Drag the 'dist' folder to netlify.com
```

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.

## Common Questions

**Q: Do I need an API key?**  
A: Not if running in claude.ai artifact. For standalone deployment, yes.

**Q: Is my data saved?**  
A: Yes, in your browser's localStorage. It persists across sessions.

**Q: Can I reset and start over?**  
A: Yes, click "Reset Business" in the dashboard header.

**Q: Can I edit the onboarding answers?**  
A: Currently no, but you can reset and start over.

**Q: How much does this cost to run?**  
A: Hosting is free. API costs are ~$10-50/month for active use.
